### **Commit Message 규칙**

| 메시지 타입 | 설명 |
| :--- | :--- |
| **feat** | 새로운 기능을 추가하거나 기존 기능을 수정할 때 사용합니다. |
| **fix** | 버그를 수정할 때 사용합니다. |
| **docs** | 문서를 수정할 때 사용합니다. |
| **style** | 코드의 스타일이나 포맷을 변경할 때 사용합니다(기능 변경 없음). |
| **revert** | 이전 커밋으로 되돌릴 때 사용합니다. |
